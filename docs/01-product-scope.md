# 产品范围

日常开发优先阅读 [Project Brief](00-project-brief.md)。本文件只补充目标用户、商品输入、风格预设和扩展构思；MVP 边界以 Project Brief 为准。

## 1. 目标用户

第一版：

- 个人开发者自己使用、演示和小范围试单验证。

潜在用户：

- 电商运营。
- 短视频剪辑团队。
- 小红书/抖音内容创作者。
- 品牌营销人员。
- AIGC 运营工具用户。

## 2. 商品输入

| 字段 | 类型 | 来源 |
| --- | --- | --- |
| 商品名 | 必填文本 | 用户填写 |
| 类别 | 下拉单选 | 后台管理 |
| 价格 | 选填文本 | 用户填写 |
| 卖点 | 多选，可选填 | 后台管理 |
| 适用人群 | 多选，可选填 | 后台管理 |
| 品牌调性 | 选填文本 | 用户填写 |
| 禁用词 | 选填文本 | 用户填写 |
| 商品图片 | 1-5 张 | 用户上传 |
| Logo | 可选 | 用户上传 |
| 背景音乐 | 可选 | 用户上传 |

商品名必须由用户填写。商品图片只作为素材输入，不作为商品资料解析入口。

当前不支持：

- 淘宝链接解析。
- 网页路径解析。
- 本地路径解析。
- 只通过图片自动识别商品资料。

## 3. 风格预设

第一批风格：

- 抖音爆款。
- 小红书种草。
- UGC 口播。
- 促销带货。
- 高级感。
- 教程演示。
- 痛点解决。
- 抽象整活。
- 搞笑反差。

每个风格必须有动态样品 GIF 或短动画展示。用户不应该只靠文字猜风格。

## 4. 扩展构思

- 竞品链接和竞品截图分析。
- 真人出镜模式。
- 30 秒视频。
- 用户自定义镜头数量和每镜头时长。
- 精准字幕对齐。
- 自动音乐推荐。
- 多平台文案发布。
- 批量生成多个商品视频。
- 视频编辑模型用于局部修复、风格统一和二次修改。
